# 📝 To-Do List Web App 

A simple and responsive To-Do List application built using **HTML, CSS, and JavaScript**, with support for **local storage** to persist tasks even after refreshing the browser.

---

## 🔗 Live Demo

👉 https://mano-dev-code.github.io/TO_DO_LIST-/

---

## 💻 GitHub Repository

👉 https://github.com/Mano-dev-code

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✔️ Mark tasks as completed
* 💾 Data stored in browser (Local Storage)
* 🔄 Tasks persist after page reload
* 📱 Responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/Mano-dev-code/TO_DO_LIST-.git
```

2. Open the project folder:

```
cd TO_DO_LIST-
```

3. Run the app:

* Open `index.html` in your browser
